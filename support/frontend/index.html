Human-in-the-Loop Approval Flow 

sequenceDiagram
    participant User as üë§ User
(VS Code)
    participant Orchestrator as üéØ Orchestrator
    participant RiskAssessor as ‚ö†Ô∏è Risk Assessor
    participant HITLManager as üõ°Ô∏è HITL Manager
    participant LangGraph as üîÑ LangGraph
Workflow
    participant LinearClient as üìã Linear Client
    participant LinearUI as üñ•Ô∏è Linear UI
(DEV-68 Hub)
    participant PostgreSQL as üóÑÔ∏è PostgreSQL
Checkpoints

    User->>Orchestrator: Submit Task
    Orchestrator->>RiskAssessor: Assess Risk Level
    
    alt Low Risk
        RiskAssessor-->>Orchestrator: Auto-Approve
        Orchestrator->>LangGraph: Execute Task
    else High/Critical Risk
        RiskAssessor-->>Orchestrator: Requires Approval
        Orchestrator->>HITLManager: Create Approval Request
        HITLManager->>PostgreSQL: Store approval_requests
        PostgreSQL-->>HITLManager: request_id
        
        HITLManager->>LangGraph: Set approval_request_id
        LangGraph->>PostgreSQL: Save Checkpoint
        LangGraph->>LangGraph: Interrupt at approval_gate
        
        HITLManager->>LinearClient: Create Sub-Issue
        LinearClient->>LinearUI: POST to DEV-68
        LinearUI-->>LinearClient: Issue Created (PR-XXX)
        LinearClient-->>User: Notification (< 1s)
        
        User->>LinearUI: Review Request
        User->>LinearUI: Set Request Status
(Approved/Denied/More Info)
        LinearUI->>LinearClient: Webhook Trigger
        LinearClient->>HITLManager: Update Approval Status
        HITLManager->>PostgreSQL: UPDATE approval_requests
        
        alt Approved
            HITLManager->>LangGraph: Emit Resume Event
            LangGraph->>PostgreSQL: Load Checkpoint
            LangGraph->>LangGraph: Resume from approval_gate
            LangGraph->>Orchestrator: Continue Execution
            Orchestrator->>User: Task Completed
        else Denied
            HITLManager->>LangGraph: Emit Cancel Event
            LangGraph->>User: Task Cancelled
        else More Info Required
            HITLManager->>User: Request Justification
            User->>LinearUI: Add Comments
            LinearUI->>LinearClient: Updated Context
            LinearClient->>LangGraph: Resume with Context
        end
    end
