# Icon Placeholder

This file serves as a placeholder. Replace with a 128x128 PNG icon.

**Design Guidelines:**

- Size: 128x128 pixels
- Format: PNG with transparency
- Theme: Match VS Code's design language
- Colors: Use brand colors (blue/purple gradient recommended)
- Elements: Consider incorporating:
  - Dev-Tools logo
  - Agent/orchestration motif
  - Chat bubble indicator

**Tools to Create Icon:**

- Figma: https://figma.com
- Canva: https://canva.com
- GIMP: https://gimp.org
- Icon generators: https://icon.kitchen

**Naming:**
Save as `icon.png` in this directory.

**Example SVG to PNG Conversion:**

```bash
# If you have ImageMagick
convert icon.svg -resize 128x128 icon.png

# Or use online converter
# https://svgtopng.com
```
